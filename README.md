# Fed News

Development updates for the [$FED project](https://github.com/snark-tank/ralph) on Solana.

## What is $FED?

Autonomous USD1 distribution system. Collects trading fees, distributes stablecoins to holders every 2 minutes.

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Fed Loop

Continuous AI development using the Ralph technique.

```bash
./scripts/fed-loop.sh
```

Files:
- `PROMPT.md` - Loop instructions
- `BACKLOG.md` - Task queue
- `CHANGELOG.md` - History

## Stack

- Next.js 16
- TypeScript
- Tailwind CSS
